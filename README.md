# AI Keyword-Based URL Downloader

A Python project that uses AI-style keyword expansion + web search
to find relevant URLs and download their contents automatically.

## Features
- Keyword expansion (LLM-ready)
- Web search (DuckDuckGo)
- URL filtering
- File & page downloading
- Clean modular architecture

## Use Cases
- Research automation
- Dataset collection
- OSINT-style crawling
- AI-powered scraping pipelines

## Run
```bash
pip install -r requirements.txt
python src/main.py
